NZBRunner by maesse

Simple tool that will move .nzb files to a desired folder.

NZBRunner is a (very) simple tool that is useful when you don’t want to waste time moving those .nzb files to a watchfolder. When you run it the first time (or without arguments), you will be presented with a simple GUI where you can select the folder you want the files to be move to. You can also enable AutoRename, so you won’t have to bother when you have conflicting filenames. Patches are welcome. :) Enjoy -Mads Lind

Dependencies

.NET Framework 2.0

Install

1. Run the installation file 2. Run NZBRunner once either directly or by running a .nzb file 3. Select a default destination folder 3a. Optimal: Select the auto rename file options 4. Hit Apply
